# Waste Management System Statement

## Problem Statement
The current waste management system is inefficient and disorganized, leading to environmental pollution and health hazards.

## Scope of the Project
This project aims to develop a simple and user-friendly waste management system for tracking and managing waste collection.

## Target Users
The target users of this project are waste management companies, municipalities, and individuals who want to manage their waste effectively.

## High-Level Features
- Waste Tracking: Track wet and dry waste collection
- Data Management: Store and manage waste data in a database
- User Interface: Simple and intuitive GUI for easy use
